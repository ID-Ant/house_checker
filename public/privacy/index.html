Calculator About How It Works FAQ Privacy Terms

Calculator inputs

The calculator uses the information you enter in the browser to estimate a starting offer. Inputs such as asking price, selected repairs, and property details are used to generate the on-page result.

Shareable links

If you create a shareable link, the page stores the chosen settings in the URL so the same setup can be reopened later. Only include information in a shared link that you are comfortable sending to someone else.

External data sources

The site uses HM Land Registry related data to support market context and property history lookups. Third-party data sources may have their own usage terms and privacy notices.

Project feedback

Feedback and issue reporting are currently routed through the public project repository on GitHub. If you submit a report there, your GitHub account information and anything you include in your report will be handled through GitHub's platform.
